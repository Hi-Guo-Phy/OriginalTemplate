# LectureNote_Template

## 简单介绍
本仓库是一个简洁的Latex模板，主要用于日常笔记写作。

## 模板架构
```text
├── content/
│   ├── chapter1.tex
│   └── chapter2.tex
├── images/
├── code/
├── mynotes.sty
├── main.pdf
├── main.tex
└── README.md
```
- ```content``` 文件夹中存放每一个章节对应的 tex 代码
- ```images``` 文件夹中存放插入的图片
- ```code``` 文件夹中代码原始文件
- ```mynotes.sty``` 文件中为模板的整体设置
- ```main.tex```为主文件
- ```main.pdf``` 为输出
